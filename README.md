# NestCheck DXF/SVG v2.5.0

NestCheck DXF/SVG is a Windows tool for importing, checking, nesting, and exporting DXF/SVG laser-cut parts. It supports material sheets, saved remnant material, machine-specific SVG/DXF export profiles, and since v2.5.0 also SVG files with embedded engraving images plus cuttable outer contours.

## Highlights

- SVG/DXF import for laser workflows
- Nesting on full sheets and saved remnant material
- Export profiles for Trotec, LightBurn/RDWorks, Epilog/Universal, xTool/Generic
- Support for engraving motifs with embedded raster image and cuttable frame
- Multilingual Windows application with trial/license workflow
- Nuitka-based Windows installer
